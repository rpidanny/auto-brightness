# auto-brightness
Change the Brightness of your Display according to the ambient light intensity.

Pre Requireents:
---------------------
 - Node.js
 - ddccontrol
 - xbacklight
 - i2c-dev
 
Installing Dependencies:
```text
sudo ./install.sh
```
Running Host Programt:
```text
sudo ./AUTO-BRIGHTNESS [options]
```
Options :
 - serial
 - hid
